# Telegram Mini App

Это мини-приложение для Telegram, созданное с использованием React.

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm start
```

## Сборка для продакшена

```bash
npm run build
```

## Технологии

- React
- Telegram Web App SDK
- Create React App
